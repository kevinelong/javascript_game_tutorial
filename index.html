JavaScript Game Tutorial 


Each link is to a demo of game features being added.
Code emphasis is on JavaScript prototypal inheritance.
A basic understanding of classical inheritance is assumed.

game01.html
A place to play and a player.
Create a Player and move it to a random position. Introduces: Sprite Class - A visual representation of an object in the world e.g. A Player. World Class - A place to play. Implements Interactivity(Arrows) and allows Sprites to move inside.

game02.html
Player extends Sprite to define a kind of Sprite Game extends world to become a kind of world that has a list of players.

game03.html
Adds a second player. Ship extends player to add a border and a collided method. Arena extends Game to add isCollided collision detection.

game04.html
Displays collisions. Introduces the Game Loop.

game05.html
HeavyShip Adds Momentum. EnemyShip distinguishes itself from player's HeavyShip. Uses gameLoop to apply momentum over time.
